# ProtectWife
C++写的安卓手游
=
利用的是cocos2d-x游戏引擎
-
算是对塔克大战的改进版本吧 
1.地图可以移动（很大） 
2.增加媳妇元素 
3.改变玩法（通关条件） 
4.增加3个大招 
5.可以上传分数 
6.增加商店功能
